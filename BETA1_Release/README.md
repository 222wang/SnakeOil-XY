## Parts printing

- Material : PETG, ABS, PC or anything you think will withstand the chamber temp you need.

- Recommended slicer setting:
  - Layer height : 0.2mm
  - Wall lines count : 5
  - Top/Bottom layers : 5
  - Infill : Gyroid 30%
  - Support placement : Touching builplate only
